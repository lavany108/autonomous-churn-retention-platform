# autonomous-churn-retention-platform
Autonomous AI platform that predicts customer churn and delivers intelligent retention actions.

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: React
- **Database**: PostgreSQL
- **ML Libraries**: scikit-learn, pandas, numpy, xgboost
- **Deployment**: Localhost

## Features

- Multi-model ML pipeline (Logistic Regression, Random Forest, XGBoost)
- Automated data preprocessing and prediction
- Intelligent retention recommendation engine
- REST API for predictions and training
- Interactive React dashboard
- PostgreSQL database for history tracking
